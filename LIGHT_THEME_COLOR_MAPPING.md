# Light Theme Color Mapping

## Current Dark Theme → Proposed Light Theme

### Background Colors
- `bg-zinc-950` / `bg-black` → `bg-white` / `bg-gray-50`
- `bg-zinc-900/50` → `bg-gray-100` / `bg-white`
- `bg-zinc-900/95` → `bg-white/95` / `bg-gray-50/95`

### Text Colors
- `text-white` → `text-zinc-900` / `text-black`
- `text-zinc-300` → `text-zinc-700` / `text-zinc-600`
- `text-zinc-400` → `text-zinc-600` / `text-zinc-500`
- `text-zinc-500` → `text-zinc-600` / `text-zinc-700`

### Border Colors
- `border-zinc-800` → `border-gray-200` / `border-gray-300`
- `border-zinc-700` → `border-gray-300` / `border-gray-400`

### Card/Container Backgrounds
- `bg-zinc-900/50` → `bg-gray-50` / `bg-white`
- `bg-red-500/10` → `bg-red-50` (lighter red tint)
- `bg-emerald-500/10` → `bg-emerald-50` (lighter green tint)
- `bg-yellow-500/10` → `bg-yellow-50` (lighter yellow tint)
- `bg-cyan-500/10` → `bg-cyan-50` (lighter cyan tint)
- `bg-blue-500/10` → `bg-blue-50` (lighter blue tint)

### Navigation Bar
- `bg-zinc-900/95` → `bg-white/95` / `bg-gray-50/95`
- `border-zinc-800` → `border-gray-200`
- `text-zinc-400` → `text-zinc-600`
- Active state: Keep `text-emerald-400` (works on light background)

### Buttons
- Primary buttons: Keep gradient (emerald-500 to cyan-500) - works on light
- Secondary buttons: `bg-zinc-800` → `bg-gray-200` with `text-zinc-900`
- "CHECK VIBE" button: Already white, change to dark variant or keep

### Score Gauge
- Background circle: `text-zinc-900` → `text-gray-200` (lighter)
- Score colors: Keep same (emerald, cyan, etc.) - they work on light

### Special Elements
- Streak badge: `bg-zinc-900/50` → `bg-gray-100` with `border-gray-300`
- Loading spinner: Keep emerald-400 (works on light)
- Error states: Keep red-400 (works on light)

## Visual Impact

**Light Theme Benefits:**
- Cleaner, more modern appearance
- Better readability in bright environments
- More professional/corporate feel
- Easier on eyes for extended use

**Considerations:**
- Gradient text might need darker variants for contrast
- Some shadows might need adjustment
- Glow effects might need to be more subtle
